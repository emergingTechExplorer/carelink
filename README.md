# CareLink

Connecting parents with trusted babysitters

## Tech Stack

### Front end 

- React.js
- Chakra UI

### Backend 

- Spring Boot
- Spring Security
- H2 Database

## Run for development

### Prerequisites

- Java (version 11 or higher)
- Maven (version 3.6.x or higher)

### Frontend

```
cd carelink/frontend
npm install 
npm run dev
```

### Backend

```sh
cd carelink/backend 
mvn spring-boot:run 
```

### H2 console access URL 

```sh
http://localhost:8085/h2-console 
```
