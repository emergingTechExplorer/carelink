# CareLink

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Chakra UI v3.x

## Setup `.env` file

```sh
MONGO_URI=your_mongo_uri
PORT=5000
```

## How to run 

### For Production

```
cd carelink
npm run build
npm run start
```

### For Development 

```sh
cd carelink
npm run dev
cd carelink/frontend
npm run dev
```
